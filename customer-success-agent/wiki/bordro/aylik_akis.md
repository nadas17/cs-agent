# Aylık Bordro Akışı
_Kaynak: System prompt Bölüm 5.3 | Son güncelleme: 2026-04-05_

## Özet
Aylık bordro (lista płac) süreci ay sonundan başlayarak ZUS beyanname gönderimine kadar sürer. Çalışma saati verisi ay sonundan 3 gün önce teslim edilmelidir.

## İçerik

### Bordro Takvimi

| Tarih | İşlem |
|-------|-------|
| Ay sonu - 3 gün | Çalışma saati verisi teslimi |
| Ay sonu - 2 gün | Ewidencja czasu pracy (çalışma süresi kaydı) |
| Ay sonu - 1 gün | Prim/komisyon bilgileri |
| Ay sonu + 7 gün | Maaş hesaplama tamamlanır |
| Ay sonu + 12 gün | ZUS beyanname gönderimi |
| Ay sonu + 10 gün | Bordro raporları teslim |

### Bordro Sorumlusu
Tüm bordro işlemleri **Payroll Specialist** (Bordro Uzmanı / Payroll Specialist) tarafından yürütülür. Bordro ile ilgili sorular Payroll Specialist'ya yönlendirilmelidir.
