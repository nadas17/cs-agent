# Vekâletname Seti
_Kaynak: System prompt Bölüm 4.1 | Son güncelleme: 2026-04-05_

## Özet
Yeni müşteri onboarding sürecinde şirket türüne göre (Sp. z o.o. veya JDG) farklı vekâletname setleri gereklidir. UPL-1 her iki tür için de ilk tamamlanması gereken formdur.

## İçerik

### Sp. z o.o. (Limited) için Vekâletname Seti

| Form | Açıklama | Zorunluluk |
|------|----------|-----------|
| UPL-1 | Elektronik beyanname vekâleti | ✓ Zorunlu — İLK TAMAMLANMASI GEREKEN |
| KAP | Genel muhasebe vekâleti | ✓ Zorunlu |
| KAO | Elektronik işlem yetkilendirme | ✓ Zorunlu |
| ZPA | Çok ortaklı firma kayıt bildirimi | ✓ Zorunlu |
| PPO-1 | Genel vekâletname – CRPO | ○ Opsiyonel |
| ZUS PEL | SGK vekâleti | ✓ Zorunlu (çalışan varsa) |
